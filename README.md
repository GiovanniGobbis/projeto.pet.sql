# Projeto SQL PetShop

Este repositório contém todos os scripts SQL utilizados no projeto de banco de dados do PetShop.

## Conteúdo
- create_tables.sql → Criação das tabelas
- inserts.sql → Inserção dos dados
- selects.sql → Consultas SELECT
- updates.sql → Atualizações UPDATE
- deletes.sql → Exclusões DELETE

## Como usar
1. Execute create_tables.sql no SQLite.
2. Execute inserts.sql.
3. Utilize selects/update/delete conforme a atividade.

Projeto finalizado para entrega acadêmica.
